Server access and the data sets for the first project

We have chosen two data sets for your first project:

Iris Data Set: http://archive.ics.uci.edu/ml/datasets/Iris
Pima Indians Diabetes Data Set: https://archive.ics.uci.edu/ml/datasets/Pima+Indians+Diabetes

You can download the csv file from the UCI machine learning repository to try solely in R. If you want to proceed and try it in SciDB+R, we have already loaded those two data sets in our SciDB server. The arrays are named "iris" and "diabetes".

Please note that in the Iris data set the target class is represented by a string. For convenience purposes when we imported the Iris data into SciDB, we replaced the target class with integers starting from 0:

Old target       New target
Iris-setosa               0
Iris-versicolor           1
Iris-virginica            2

In order to finish the project, first you need to install R on your own computer. The version I recommend is R 3.2.3.
The download links are provided below:

Windows: https://cran.r-project.org/bin/windows/base/
Mac OS X: https://cran.r-project.org/bin/macosx/

If you are using Linux operating systems such as Ubuntu, you should refer to https://cran.r-project.org/bin/linux/ubuntu/README for more information.
Once you have R correctly installed, you can run the following commands to install the SciDB-R package:

install.packages('devtools')
devtools::install_github("Paradigm4/SciDBR")

The second command may take longer in Mac OS X and Linux. Please note that if you are using R on Windows platforms, you also need to install Rtools in addition to the R binaries in order to be able to run the commands above.

The SciDB-R library can be loaded by running:

library('scidb')

To connect to our SciDB server, run:

scidbconnect('[server address]',8080L)

The server address is now available both on and off campus.

A brief introduction to SciDB-R can be found here: http://www.paradigm4.com/try_scidb/scidb-r/
Below is a code snippet to connect to the SciDB server using the R library and compute the Gamma matrix on the Iris data set:

library('scidb')
scidbconnect('[server address]',8080L)

arrName="iris"
strquery = paste("DenseGamma(", arrName, ");", sep="")
Gamma_vertical = iquery(strquery, return=TRUE)
d = max(Gamma_vertical$i)
Gamma = matrix(0, nrow=d, ncol=d)
k = 1
for (i in 1:d)
for (j in 1:d) {
  Gamma[i,j] = Gamma_vertical[k,3]
  k = k+1
}

You can also use the web interactive SciDB console at http://[server address]:8080/query.html to play with the SciDB query language (AFL and AQL).
